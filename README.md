# El Baúl de la Abuela

Sitio web oficial de El Baúl de la Abuela — arriendo de trajes de época, vestuario y accesorios para sesiones fotográficas y eventos.

Sitio publicado en: https://elbauldelaabuela.net

## Tecnologías

Sitio estático: HTML + CSS. Sin dependencias ni build process.

## Estructura

- `index.html` — página principal
- `*.html` — páginas de categorías y servicios (trajes, sesiones, precios, contacto…)
- `styles.css` — estilos globales
- `imagenes/` — recursos gráficos del catálogo
